# ImgToFont
## 이미지 -> 폰트 생성기
이미지를 이용하여 폰트를 생성하는 프로그램을 구현한다.
> 해당 프로그램은 [potrace]를 사용하여 구현하였습니다.<br>
> 윈도우 사용자는 'potrace.exe'를 직접 다운로드하여 루트 폴더에 넣어주세요.
>  1. [potrace 공식 다운로드 페이지](http://potrace.sourceforge.net/#downloading) 접속
>  2. Windows - zip 파일 다운로드
>  3. zip 파일의 압축을 풀고 potrace.exe 파일을 루트 폴더에 복사



### 기능
- [x] 한글 자모, 음절 규칙 정의
- [ ] PNG -> SVG 변환 기능
- [ ] 글리프 생성 기능
- [ ] 폰트 파일 생성 기능
- [ ] 기본 데이터 구축

### 입출력
#### 입력
- [ ] PNG 폴더 경로
- [ ] SVG 폴더 경로
- [ ] Font 경로
- [ ] Font 이름 지정

#### 출력
- [ ] 폰트 빌드 단계별 진행 상황 출력
- [ ] 예외 상황 시 에러 문구를 출력

### 예외 처리
- [ ] 폰트 빌드 오류 출력
- [ ] 글리프 빌드 오류 출력
- [ ] PNG -> SVG 변환 오류 출력
